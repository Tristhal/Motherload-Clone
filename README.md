# Motherload-Clone
A simple python game inspired by the flash game "Motherload" made using pygame. The goal is to make 1 dollar.
This is a simple game created for a high school project. It is a 2D game where you dig downwards deeper and deeper to find mythical substance which you must obtain in order to earn your first dollar. Along the way you will find various resources which allow you to upgrade your mining ship (square) along your journey to the bottom of the world.
